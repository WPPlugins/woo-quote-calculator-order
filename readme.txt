== Description ==

Woo Quote Calculator & Order creates dynamic quote form and add quote results at shopping cart with one click 

### The features of the slider

Woo Quote Calculator and Order Makes simple responsive qoute form and calculate quote result for customers.
It Produces number of products they need to buy dynamically based on the user imput and quote formula. 
Customer can buy all the products generated from a quote with a single click. 
They can request a quote , get result and order the products all in one page, just using simple shortcode in post/page or widgets. 

* Easy to use
* Create  dynamic & smart quotes with  GUI
* See quote result and order at same page, using ajax
* Order all calculated products to shopping cart at one click
* Fully responsive quotes, results and admin interface
* Possible to create unlimited number of quotes and products
* One click export and import any/all  quotes
* Documentation included within plugin menu for easy help

== Installation ==

### First download the ZIP file ,

You can either install it automatically from the WordPress admin, or do it manually.

1. Unzip the archive and put the `woo-quote-calculator-and-order` folder into your plugins folder (/wp-content/plugins/)
2. Activate the plugin through the `Plugins` menu in WordPress
3. You should now see Woo Quotes menu at your dashboard
